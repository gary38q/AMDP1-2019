How to find the first non repeated character of a given string ?

The answer her is d. So the point that strikes me is that it can be done in two ways : For every index i loop i-1 times and check if that character occurs ever again. But this is not efficient : growth of this method is O(N^2) where N is the length of the string. Another possible good way could be if I could form a trere or any other ds such that i could order the character based on the weights (the occurrencs count ). This could take me just loopo of length N through the string to form the structure. That is just O(N) + O (time to build tree or any ds).

Here's a very straightfoward O(n) solution: def fn(s): order = [] counts = {} for x in s: if x in counts: counts[x] +=1 else: counts[x] = 1 order.append(x) for x in order: if counts[x] == 1: return X return None We loop through the string once. When we come across a new character, we store it in counts with a value of 1, and append it to order. When we come across a character we've seen before, we increment its value in counts. Finally,we loop through order until we find a character with a value of 1 in counts and return it.

A list comprehension will give you the characters in the order they appear if they onlu appear once: in[61]: s = 'aabccbdcbe' in [62]: [a for a in s if s.count(a) == 1] Out[62]: ['d','e'] Then just return the first entry of this: in [63]:[a for a in s if s.count(a) == 1][0] Out [63]: 'd' if you just need the first entry, a generator would work as well: in [69]: (a fora in s if s .count(a) == 1).next() Out[69]:'d'
